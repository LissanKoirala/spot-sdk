API Example - Using the Timesync Service

This example demonstrates how to use the timesync service to establish time sync between your computer and the robot's clock. Specifically, it creates a TimeSyncEndpoint, which can be used to establish timesync as well as determine the clock skew or round trip time.

Setup Dependencies

This example needs to be run with python3, and have the SDK 2.0 installed. See the requirements.txt file for a list of dependencies which can be installed with pip using the command:

python -m pip install -r requirements.txt

Running the Example

To run the examples:

python time_sync_client.py --username USER --password PASSWORD ROBOT_IP
