API Example - World Object Mutations

These examples demonstrate how to use the world object service to add, change, and delete world objects.

## Setup Dependencies
These examples need to be run with python3, and have the SDK 2.0 installed. See the requirements.txt file for a list of dependencies which can be installed with pip.
```
python -m pip install -r requirements.txt
```

## Running the Example
To run the example:
```
python mutate_world_objects.py --username USER --password PASSWORD ROBOT_IP
```
